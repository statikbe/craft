# Statik's Craft Base Install

Welcome to the comprehensive documentation for [Statik's](https://www.statik.be) Craft CMS base install. This documentation covers our custom frontend components, Twig extensions, and development tools to help you build powerful, accessible web applications.

40+ reusable JavaScript components for common UI patterns including:

- **Form Components**: Validation, autocomplete, datepickers, matrix fields
- **Interactive Elements**: Modals, dropdowns, tabs, accordions, tooltips
- **Advanced Features**: AJAX search/paging, maps (Google & Leaflet), Swiper sliders
- **Media Components**: Video backgrounds, parallax effects, masonry layouts

### [Clint CLI Tool](/frontend/clint/introduction.html)

Command-line interface for managing Craft CMS frontend updates and tests.

### [Craft & Twig](/craft/about.html)

Custom Twig extensions and utilities for Craft CMS:

- **Icon Extension**: SVG icon rendering with accessibility features
- **Hyper Extension**: Verbb Hyper field link rendering
- **Hyphenate Extension**: Multi-language text hyphenation
- **Paginate Extension**: Customizable pagination UI
- **Statik Extension**: SEO control, bot detection, slugification
- **Validate Input Extension**: Security-focused input validation
